# BariBari Server

기존 내 코스만 저장하는 바리바리 앱을 개선합니다.

기존 앱은 쌍둥이가 만들었는데, 서버 붙이는 거 아직 허락 못 받았습니다.


---

## 1차 MVP — 라이딩 코스 공유 피드

라이딩 코스를 서로 공유하는 커뮤니티 서버.

늘 달리던 곳만 달려서 지겹기도 하고,  
다른 라이더가 알고 있는 좋은 코스가 없을까 하는 의문에서 시작했습니다.

### 기능 명세

- 내 코스를 다른 사람들에게 공유할 수 있습니다.
- 다른 사람들이 공유한 코스를 구경할 수 있습니다.
